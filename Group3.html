HOW TO HOST STATIC WEBSITE ON S3 USING CLOUDFRONT.

Hello, we are going to demonstrate how to host a static website on s3 and CloudFront.

STEPS TO FOLLOW:

  1. Create an S3 Bucket 
  2. Upload the Static content making object permission public
  3. Enable Website Hosting on the Bucket
  4. Make the bucket Publicly Accessible
  5. Configure Index Document
  6. Test the Static Web Endpoint
